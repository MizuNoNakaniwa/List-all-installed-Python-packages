# pyrecords — 一键盘点 Python 下载/安装记录

扫描你电脑上所有 Python 相关的下载和安装痕迹，汇总成一份带时间线的清单，自动存到桌面（`Python下载记录.txt`）。

Python 本身没有统一的"下载历史"文件，这个工具从各处痕迹把账本拼出来，一次告诉你：

- **[0]** 电脑上装了几个 Python（含 py launcher、PATH、conda/Miniforge 各环境）
- **[1]** 每个 Python 装过哪些包、什么时候装的（按时间倒序的流水账）
- **[2]** pip 下载缓存里留存的安装包（就算卸载了，缓存底子还在）和缓存占了多少 MB
- **[3]** conda 的安装/卸载历史（逐条命令 + 时间，最接近真账本）
- **[4]** 模型缓存里各模型的下载时间和大小（HuggingFace / ModelScope / PaddleOCR / EasyOCR）

## 使用方法

1. 下载 `find_records.ps1` 和 `run.bat`，放进同一个文件夹
2. 双击 `run.bat`，等待扫描完成（大概一分钟）
3. 结果会打印在窗口里，同时保存到 **桌面的 `Python下载记录.txt`**
4. 想让 AI 帮你分析的话，把 txt 内容贴给它就行：啥时候下了啥、哪些是废的、缓存能不能清

## 说明

- 仅支持 Windows，不需要装任何额外东西（用的是系统自带的 PowerShell）
- 纯只读扫描：不删除、不修改任何文件，放心跑
- 没装 Python 也能运行，只是会报告"没检测到 Python"
- 每个解释器的包列表最多显示 120 条（按时间最近的排前面），够用了
- 如果杀毒软件拦截，是因为 bat 调用了 PowerShell，允许即可
